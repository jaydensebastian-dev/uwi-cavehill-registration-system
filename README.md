## UWI Cave Hill Registration System (C++)

### Description
A console-based system that simulates student registration at UWI Cave Hill. Users can add students, add courses, enroll students in courses, and generate schedules. Data is saved to files so the system can be reused.

### Features
- Add / view students
- Add / view courses
- Enroll a student in a course (with validation)
- View a student's enrolled courses
- Save & load data from files
- Search and sort students/courses

### Tech Stack
- C++

### How to Run
1. Open in any C++ IDE (Visual Studio / CodeBlocks) or use terminal
2. Compile:
   `g++ -std=c++17 main.cpp Student.cpp Course.cpp Registry.cpp -o app`
3. Run:
   `./app`
